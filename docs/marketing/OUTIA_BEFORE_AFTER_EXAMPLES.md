## CONVERSION COPYWRITING CHECKLIST

Before publishing ANY copy, ask:

- [ ] Does it START with a benefit, not a feature?
- [ ] Could a stranger understand it in 5 seconds?
- [ ] Does it use SPECIFIC numbers (not vague words)?
- [ ] Does it ACKNOWLEDGE the user's pain point?
- [ ] Does it show HOW it solves that pain?
- [ ] Does it include SOCIAL PROOF?
- [ ] Does it have a CLEAR CTA?
- [ ] Could it be SHORTER?
- [ ] Is it CONVERSATIONAL (not corporate)?
- [ ] Does it make user feel SMART/CAPABLE/RECOGNIZED?
